# Описание
1. Создать класс матрицы построенного на шаблонах (подразумивается, что тип будет элементом поля c дефолтным контруструктором, конструктором копий)
2. Создать класс системы линейных уравнений и решить их с помощью предыдущего пункта 
# Подробности
Класс матрица должен поддердивать следующие операции:
1. Сложение (+)
2. Умножение (+)
3. Транспонирование (+)
4. Поиск обратной (+)
5. Приведение к ступенчатому виду (+)
6. Поиск определителя (+)
7. Ранг матрицы (+)
8. Обёртка над системой (Сделана функцией solve)
9. Поиск собтвенных чисел и векторов (поиск спектра)
10. Приведение к жордановой форме
11. Кронокерово произведение
12. Разложение на различные виды (LUP SVD)
